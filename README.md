# 🩸 Blood Donation App - Oman

A mission-driven mobile application designed to bridge the gap between blood donors and recipients across the Sultanate of Oman. Built with **Flutter** and **Firebase**, this app aims to make blood donation faster, more accessible, and life-saving.

---

## 🌟 Key Features

* **📍 Smart Search:** Find nearby donors based on blood type and current location.
* **🚨 Emergency Requests:** Send instant alerts to potential donors when life-saving blood is needed immediately.
* **📝 Donor Registration:** Easily register as a donor and manage your availability.
* **📜 Donation History:** Track your past donations and stay notified on when you are eligible to donate again.
* **🏥 Blood Bank Locator:** Integrated maps to find and navigate to the nearest blood banks or donation events.
* **📅 Event Management:** Stay updated on blood drives and community events across Oman.

---

## 🚀 Tech Stack

* **Frontend:** [Flutter](https://flutter.dev/) (Multi-platform Android & iOS)
* **Backend:** [Firebase](https://firebase.google.com/) (Authentication, Firestore, Cloud Messaging)
* **Maps:** Google Maps API / Location Services
